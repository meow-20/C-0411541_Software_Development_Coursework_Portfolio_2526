# Task Management System (Command-Line Application)

## Overview
This is a simple **command-line task management system** developed in **Python**.  
The application allows users to create, view, manage, and store tasks persistently using file I/O.

This project was developed as part of a **summative assessment** and demonstrates user interaction, data storage, error handling, and structured code design.

---

## Structure
Task-Management-System/
├── main.py
├── README.md
└── tasks.json


## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Assign deadlines to tasks (validated format)
- Assign priorities to tasks
- Sort tasks by deadline
- Sort tasks by priority
- Persistent storage using a JSON file
- Input validation and error handling

---

## Technologies Used
- Python 3
- JSON (for data storage)
- Command-line interface (CLI)
